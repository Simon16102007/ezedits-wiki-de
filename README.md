## Was ist ezEdits?

ezEdits ist ein WorldEdit/FastAsyncWorldEdit (WE/FAWE) erweiterndes Plugin, das zahlreiche neue Befehle, Muster, Masken und Werkzeuge hinzufügt, mit diesen erstaunliche Dinge erschaffen werden können!
